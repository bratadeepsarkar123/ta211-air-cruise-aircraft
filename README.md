# TA211 — "Air Cruise" Scaled Conceptual Aircraft

Course project, TA211 (Manufacturing Processes I), IIT Kanpur — Prof. Shashank Shekhar, Group F14 (Aug – Nov 2025).

Scaled physical model of a multi-wing conceptual aircraft ("Air Cruise") with a segmented fuselage, panoramic dome, 16 engine pods, and modular landing gear. Designed in Autodesk Inventor with full fabrication drawings (isometric + orthographic) and workshop execution (MS sheet cutting/bending, cast-iron casting for dome and wheels, tube cutting).
